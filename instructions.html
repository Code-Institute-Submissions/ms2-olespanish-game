¡Olé Spanish!

Instructions

¡Olé Spanish! memory game is a great way to practice Spanish vocabulary. this memory game will test both your vocabulary and your memory.

The purpose of the game is to find all the matching pairs of cards. To do that, you first have to click on a card to burn it over, and then a second card to check if they match. If they do not match, the cards are turned back over. The game is over when the player has matched all of the pairs and all the cards are gone. Just press the "play" to start playing!

There are 3 levels of difficulty:

Beginners

In this level, every card provides 3 different pieces of information: a picture, a name and a pronunciation clip.

Intermediate

Here the game gets a little bit more complicated by adding more cards to the table.

Advanced

This level is only for masters, will you be able to match all the cards without the images help?

If you'd like to refresh the lesson's vocabulary before starting to play, just click on the "refresh vocabulary" here or on the main menu.
